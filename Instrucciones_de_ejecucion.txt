Instrucciones para la ejecución del programa:


PRIMER OPCION (UNICAMENTE USANDO WINDOWS)

EJECUCIÓN DEL CÓDIGO CON VISUAL STUDIO CODE.

Paso 1. Por medio del Classroom o por el enlace de GitHub (por la sección de la Primer Tarea) tendrá que guardar el archivo con la extensión de ".py" en algún directorio de su equipo de computo, por defecto al momento de la descarga del archivo, este se llamará "Buscador_de_nodos.py", en cambio; puede cambiar el nombre del archivo (aunque es opcional)

Paso 2: Abre Visual Studio Code.

Paso 3: Verifica si Visual Studio Code tiene la extensión de Python instalado (en caso que lo tenga instalado, puede omitir este paso). En caso de que no, en la barra lateral, en donde se muestra un icono de cuadrados, se presiona y aparece el navegador de extensiones de allí busca la extensión de Python y seleccione para poder descargarlo.

Paso 4: En la barra superior presione "File" y seleccione la opción "Open File...", este aparecerá una ventana emergente, de allí hay que navegar hasta encontrar el directorio donde usted guardo el archivo que se llama "Buscador_de_nodos.py" (o el nombre que le dio en su descarga)

Paso 5: Una vez abierto el archivo, le damos ejecutar o correr presionando el icono superior derecho (es un icono de triángulo o play)

Paso 6. El programa se ejecutará en la terminal y se imprimirá los nodos 99 y 57.


EJECUCIÓN DEL CÓDIGO EMPLEANDO CON CMD


Paso 1. Verifica si en su equipo tenga instalado Python, en caso de que no puede descargarlo desde la página oficial que es: https://www.python.org/downloads/ 

Paso 2. Por medio del Classroom o por el enlace de GitHub (por la sección de la Primer Tarea) tendrá que guardar el archivo con la extensión de ".py" en algún directorio de su equipo de computo, por defecto al momento de la descarga del archivo, este se llamará "Buscador_de_nodos.py", en cambio; puede cambiar el nombre del archivo (aunque es opcional)

Paso 3. Una vez descargado el archivo, abre el explorador de archivos de Windows y navegue hasta entrar en la carpeta o directorio en donde se encuentra el archivo de python

Paso 4: Haga clic a la barra superior en donde este muestra la ruta seleccione todo el texto

Paso 5. Borra el texto seleccionado y escriba en su lugar cmd y pulse enter.

Paso 6: Se abrirá el cmd. Para ejecutar el programa, simplemente escribiremos en la consola: "py nombre_del_archivo.py"

Paso 7: El programa en Python se ejecutará en el CMD y imprimirá los nodos 99 y 57 dentro del mismo CMD.


EJECUCIÓN DEL CÓDIGO EMPLEANDO CON POWERSHELL


Paso 1. Verifica si en su equipo tenga instalado Python, en caso de que no puede descargarlo desde la página oficial que es: https://www.python.org/downloads/ 

Paso 2. Por medio del Classroom o por el enlace de GitHub (por la sección de la Primer Tarea) tendrá que guardar el archivo con la extensión de ".py" en algún directorio de su equipo de computo, por defecto al momento de la descarga del archivo, este se llamará "Buscador_de_nodos.py", en cambio; puede cambiar el nombre del archivo (aunque es opcional)

Paso 3. Una vez descargado el archivo, abre el explorador de archivos de Windows y navegue hasta llegar el nombre la carpeta o directorio en donde se encuentra el archivo de Python (Nota: No hay que entrar en el directorio o en la carpeta).

Paso 4: Haga clic derecho con shift a la carpeta o directorio

Paso 5: Aparecerá un pequeño menú de opciones, en cambio nos dirigimos a la opción en donde dice: "Abrir la ventana de Powershell aquí"

Paso 6: Se abrirá Powershell con la ruta ya establecida, ahi mismo escribiremos en la consola: "py nombre_del_archivo.py" para poder ejecutar el programa

Paso 7: El programa en Python se ejecutará en el PowerShell y imprimirá los nodos 99 y 57 dentro del mismo.


SEGUNDA OPCIÓN (USANDO CUALQUIER OS)

Paso 1. Por medio del Classroom o por el enlace de GitHub (por la sección de la Primer Tarea) tendrá que guardar el archivo con la extensión de ".py" en algún directorio de su equipo de computo, por defecto al momento de la descarga del archivo, este se llamará "Buscador_de_nodos.py", en cambio; puede cambiar el nombre del archivo (aunque es opcional).

Paso 2: En un navegador web, escribiremos con la siguiente URL: https://www.onlinegdb.com/ y ponemos enter

Paso 3: Una vez dentro del Debugger Online, nos parecerá el editor de texto, en cambio en la barra superior, seleccionaremos  el icono de flecha apuntando a la nube, por lo cual sería "Upload File".

Paso 4: Aparecerá una ventana emergente, de allí hay que navegar hasta encontrar el directorio donde usted guardo el archivo que se llama "Buscador_de_nodos.py" (o el nombre que le dio en su descarga al archivo python) para así abrir el programa

Paso 5: Una vez que se muestra el código del programa del Python, en la barra superior, en la parte en donde dice "Language" le damos un clic y seleccionaremos "Python 3" para así poder ejecutarlo

Paso 6: Para ejecutar el programa, presionamos "F9" o presionamos el botón verde que dice "Run"

Paso 7: El programa en Python se ejecutará en la consola y imprimirá los nodos 99 y 57 correspondientes.
